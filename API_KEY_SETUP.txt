🔑 Gemini APIキー設定ガイド
================================

⚠️  重要: このステップをスキップするとアプリケーションが動作しません ⚠️

## 📋 設定手順

### 1. Gemini APIキーの取得
🌐 以下のURLにアクセスしてAPIキーを取得してください：
https://makersuite.google.com/app/apikey

手順：
1. Googleアカウントでログイン
2. 「Create API Key」をクリック
3. 「Create API key in new project」を選択
4. 生成されたAPIキーをコピー

### 2. 環境変数ファイルの編集
📝 `.env`ファイルを開いて編集してください

変更前：
```
GOOGLE_API_KEY=your-gemini-api-key-here
```

変更後：
```
GOOGLE_API_KEY=AIzaSy... (あなたの実際のAPIキー)
```

### 3. 設定の確認
✅ 設定が正しくできているか確認する方法：

1. Streamlitアプリを起動
2. 「APIキー設定の確認」セクションを展開
3. 緑色のチェックマークが表示されることを確認

### ⚠️ トラブルシューティング

❌ よくある間違い：
- APIキーに余分なスペースが入っている
- 「your-gemini-api-key-here」のまま変更していない
- .envファイルが保存されていない

✅ 正しい設定例：
```bash
# コメントアウト（#）で始まる行は無視されます
GOOGLE_API_KEY=AIzaSyA1B2C3D4E5F6G7H8I9J0K1L2M3N4O5P6Q
```

### 🔐 セキュリティについて
- APIキーは他人と共有しないでください
- GitHubなどに.envファイルをアップロードしないでください
- .gitignoreファイルに.envを追加することを推奨します

================================
設定完了後、以下のコマンドでアプリを起動してください：

streamlit run rag_system.py
================================
